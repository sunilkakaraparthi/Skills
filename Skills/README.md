# Skills Console Application

This project is a simple console application named "Skills". It serves as an introductory example of a .NET console application.

## Project Structure

- **Skills.sln**: The solution file that organizes the project and its components.
- **Skills/Program.cs**: The entry point of the console application containing the `Main` method.
- **Skills/Skills.csproj**: The project file that includes configuration settings for the application.

## Getting Started

To run this application, ensure you have the .NET SDK installed. You can download it from the official .NET website.

### Running the Application

1. Open a terminal or command prompt.
2. Navigate to the project directory.
3. Use the following command to run the application:

   dotnet run
